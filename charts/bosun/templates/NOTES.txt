Bosun chart installed.

Secrets must exist in the release namespace:
  - stern-secrets
  - quak-secrets
  - github-secrets
The envelope encryption secret is generated by default (pre-install hook Job):
  - fluyt-envelope-encr-secrets
To disable generation, set `envelopeEncryption.create=false` and provide your own secret.

Quickstart secrets (non-production) are available at:
  helm/examples/quickstart-secrets.yaml

Bundled services (Postgres/Redis/Qdrant) are enabled by default.
To use external services instead, set:
  postgres.enabled=false
  redis.enabled=false
  qdrant.enabled=false
and provide URLs via values (stern.env / quak.env / stern.database).

SCM mode is mutually exclusive:
  scm.mode=github (default) uses github-secrets
  scm.mode=gitlab requires a GITLAB_PAT secret (scm.gitlab.patSecretName)
